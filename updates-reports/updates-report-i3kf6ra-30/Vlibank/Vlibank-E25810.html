Name Bedieneinheit - Tastatur/Maus
Description

Control Unit - keyboard / mouse function, in different models is a configurable help. It can be keyboard or mouse alternative. The conversion is done using a key behind the device.
The design allows people with severe motor disabilities keyboard or mouse function to use.
The clear layout ensures that access to the computer as a tool is facilitated.
When using the simulator function as a mouse, all mouse functions controlled by buttons. As one of the four arrow keys is pressed, move the cursor in the appropriate direction. In addition to the left and right mouse buttons are also double-click and select.
It is also a control unit with joystick to buy.
One can choose a device with the left or right handed operation.
(For more information, see brochure)

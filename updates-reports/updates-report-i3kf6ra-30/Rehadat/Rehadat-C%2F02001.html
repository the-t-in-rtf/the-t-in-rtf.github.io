Product Type:
Input modification for keyboard simulation and mouse simulation with single and multiple keystrokes

Areas of Application:
- allows people with severe coordination difficulties to operate keyboard and / or mouse functions with single or multiple sensors or joysticks with IBM-compatible PCs.

Features / Components:
- USB adapter for the connection of up to 6 simple sensors, which activate previously triggered actions
- Connection can be made in addition to the standard mouse or standard keyboard
Possible freely programmable functions per sensor input:
- call up any internet pages
- start any programs from the hard drive
- Open any files
- trigger any keyboard stops
- Release the mast buttons
Keyboard key functions:
- letters, special characters, function buttons, control buttons and combinations thereof
- Examples: Enter key, CTRL + C, ALT + F4, F1, ESC, TABULATOR, etc.
Mapping button functions: Click right / center / left, click and drag left, double-click, scroll wheel simulation

PC connection: USB interface

Mini keyboard with button mouse

This mini keyboard can be used in people with severely impaired movement capability of the hands as well as muscle weakness

- Hands-free operation with a straight or curved operating rod of balsa wood
- key in an arcuate arrangement
- ergonomically the Heatmap adapted

- directly without any additional software on the computer connected
- integrated connector for a second keyboard
- adjustable one-finger function
- latching for Alt, AltGr, Shift, Ctrl as unique, permanently or non-latching adjustable
- status display for latching function LED
- 4 individually adjustable time delay
- adjustable visual and audible key confirmation
- integrated button mouse function
- is also the stylus operated
- 8 directional keys for integrated button mouse function (up, down, right, left, 4 diagonal directions)

- buttons: right, left, left Hold, Double - electronic help for: Double, Click-Drag

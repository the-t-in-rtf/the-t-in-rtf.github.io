Product Type:
Word prediction

Areas of Application:
For slow writers (for example, when using head, head, etc. to enter texts) for:
- severe motion disorder
- neuromuscular disabilities (such as ALS)
- spastic and flaccid paralysis

Description:
During writing, suggestions for word completion are made on the basis of the already written text. When a suggestion is selected, it is copied into the text. This considerably reduces the number of necessary keystrokes for inputting texts.

Features / Components:
- Word prediction learns with and can be adapted to individual vocabulary
- Prediction of composite headings
- Prediction of several consecutive words possible
- Consider the grammar of the predictions
- Generic algorithm to ensure compatibility with many languages
- general dictionary with about 300,000 entries (German)
- any number of your own dictionaries can be created by yourself
- continuous adaptation of the dictionaries possible during the application
- Predictability for different programs, even when changing applications

System requirements:
- IBM-compatible PC Pentium II, 300 MHz or faster
- Win 98 / ME / 2000 / XP / Vista32
- CD drive for installation

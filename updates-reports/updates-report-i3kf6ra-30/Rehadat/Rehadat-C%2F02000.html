Product Type:
Input modification for mouse control with single and multiple buttons

Areas of Application:
DIPAX COntrolBox allows mouse control via individual buttons / sensors or joysticks parallel to the mouse for people with severe coordination difficulties.

Description:
The ClickBox is an optional add-on module for DIPAX ClickMaster. All mouse clicks (or the mouse button selected in the ClickMaster) can be triggered directly via the ClickBox.

Up to 6 simple sensors can be connected to this USB adapter via a 3.5 mm jack plug, which can be operated independently of each other. When the input 'Select' is activated, the Mapping button, which is currently selected in the ClickMaster, is always triggered. In addition, the mouse clicks displayed on the box can be triggered directly via the other 5 inputs. Users who are able to trigger an external sensor can therefore do without the dwell function (Autoklick) and trigger the mouse click selected directly in the ClickMaster via a sensor. A combination of Auto- and Sensorklick is also possible.DIPAX ClickMaster

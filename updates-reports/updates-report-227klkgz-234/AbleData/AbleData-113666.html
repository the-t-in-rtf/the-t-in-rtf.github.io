Name Keyboard Stand (Model H-38)
Description The Keyboard Stand, model H-38, is a keyboard tray and slant board designed for use by individuals with neurological or physical disabilities. This unit holds the user's keyboard at an angle such as 45 or 60 degrees, which reduces head-bobbing, where the user has to look down, then up at the monitor, then down, then up, etc. It also eliminates the accidental production of spaces by the palm of the hand, and puts the keyboard at an angle in line with the screen. The unit is lightweight and transportable, and stays in place with Stick &amp; Stuck adhesive material on the bottom.

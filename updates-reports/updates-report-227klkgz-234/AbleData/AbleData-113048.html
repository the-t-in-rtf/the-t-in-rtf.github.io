Name Textaloud
Description TextAloud is a voice output program designed for use by individuals with low vision or reading disabilities. This software uses voice synthesis to convert text into spoken audio and read text from email, Web pages (HTML), Word files, PDF files, PowerPoint, and e-books. It can also convert text to MP3 or Windows Media (WMA) files. Toolbar plug-ins are included for Internet Explorer, Firefox, and Outlook. OPTIONS: AT&amp;T Natural Voices, Acapela, or Nuance RealSpeak text-to-speech converters. COMPATIBILITY: For use on Macintosh or IBM and compatible computer systems. SYSTEM REQUIREMENTS: Windows 95/98/ME/NT/2000/XP/Vista operating system; P200 or higher processor; 64 megabytes (MB) RAM; 10 MB disk space; and a sound card.

Name Art For All Language Kit (Model Lk-Aa)
Description Art for All Language Kit, model LK-AA, is a painting and drawing activity and language tutorial program designed for use by individuals with cognitive disabilities. Concepts addressed are art tools and how they are used, colors, and counting and graphing using M &amp; Ms. The kit includes a book written with limited vocabulary and repetitive lines using Mayer-Johnson Picture Communication Symbols; an interactive computer book that combines text, pictures, speech, and sound; and a songboard, word cards, and vocabulary games. The songboard has a different outcome each time it is used.

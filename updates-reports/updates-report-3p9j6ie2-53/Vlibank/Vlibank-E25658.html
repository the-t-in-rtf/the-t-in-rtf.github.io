Name MS Agent en Word Agent
Description

MS Agent is an application of Microsoft's animated characters to speak in a web environment. There are different figures. These different movements you can do. You can text speak with or without bubble. can breaks in the reading, the read characters from speak ...
The figures can give instructions through an agent script. This Agent or Java Script can be written in Visual Basic. With the Mass freeware program you can create pages with an agent script. Here we will not go into that.
However, interest only programs that work with MS Word such as Agent and Agent Deskbot. For these latter programs can work, must first install MS Agent.
A good text to install MS Agent you think Paul Kavelaars.
Via Microsoft Download: http://www.msagentring.org/setup.aspx
Word Agent consists of a template in MS Word, with an additional tool for. Using this tool you can use a Word document directly be read by a chosen character. (= 'Agent'). You can complete the document, only the selected text or from the cursor to read. There is also a pause button provided.
Word Agent only works with MS Agent.

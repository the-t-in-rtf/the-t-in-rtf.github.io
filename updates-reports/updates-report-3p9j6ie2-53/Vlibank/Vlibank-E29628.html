basic Features

-Operating Systems: Windows 2000 (32-bit), XP (32-bit), Vista (32-bit) and 7 (32-bit)
-Dutch UI: no
-Dutch Braille table: yes
additional information
-The Duxbury Braille Translator (DBT) is a braille word processor that is being integrated into Word.
-Create the text done in Word.
-Various text formats can be handled, including Word, WordPerfect and HTML.
-After the Duxbury converted the text into Braille, it is possible to work in DBT to change the layout of the braille document.
-The user interface is in English. DBT has also been localized into French, German ... A Dutch version is not available.
-All commercially available braille printers are supported.
-Conversion of graphic elements such as photographs or drawings is not possible with DBT. Duxbury provides a free software, QuickTac: Braille graphics can be created.
-Conversions are supported for Braille mathematics to Nemeth, BAUK, UEBC and French Braille.

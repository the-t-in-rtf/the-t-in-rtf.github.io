keyboard

Areas of Application:
This keyboard series can be used in head and mouth writers, people with dysmelias, pronounced cerebral or central movement disorders or spasticity. The keyboard is available with or without numeric keypad and has latched switching functions

Features / Components:
- Key spacing compared to the standard keyboard enlarged
- directly without any additional software on the computer connected
- continuously adjustable delay time from 0 to 3 seconds
- adjustable Einfinger function
- latching for Alt, AltGr, Shift, Ctrl as unique, permanently or non-latching adjustable
- status display for latching function LED
- switchable key quit sound
- macro function for storing up to 55 functions
- connection option for second keyboard
- user-specific parameters remain

stored in the keyboard Versions:

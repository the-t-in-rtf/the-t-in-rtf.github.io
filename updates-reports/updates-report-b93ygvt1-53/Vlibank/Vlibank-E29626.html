Name Magnilink iMax 1.1
Description

basic Features
-Version: 1.1
-Designed for: Mac OS X
-Magnification range: 1.5 to 36 times
-Texts with smooth edges: no
-Colour adjustment: yes
-Image invertible: yes
-Higher resolutions supported: yes
additional information
-Imax is a magnification software for the Mac that is compatible with the magnifiers of LVI.
-The Dutch voice synthesis software Infovox iVox (EN & VL) is included with the computer and can be started.
-Magnification and speech are switched on and off via hotkey.
-All basic functions are controlled through keyboard shortcuts.
-Magnification', 'Cursor' and 'Speaking' are adjustable with a set up window.
-There are five magnification modes: round and square magnifying glass, split screen for partial magnification, screen magnification and magnification on a second monitor.
-The magnification is adjustable between 1.5 and 36 times. The contrast is then incremented automatically. This improvement can be switched off.
-There is a choice of six color schemes.
-The mouse pointer can be enlarged, while a green by the arrowhead put axes for better visibility.
-max reads all the parts for you with your mouse to navigate.
-The user can indicate that the language of the text to be read should be automatically recognized.

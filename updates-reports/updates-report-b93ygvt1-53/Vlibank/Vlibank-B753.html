Name Window-Eyes 8.0
Description

Basic Features:
- Version: 8.0
- Designed for: Windows XP, Vista, 7 and 8
- Magnification: no
- Voice support: yes
- Braille support: yes
Additional Information:
- Window-Eyes is a screen reading software for Windows 8. It makes the computer accessible through speech and Braille support.
- Dutch speech synthesis software is included, both Dutch and Flemish voices are available.
- The speech output can be set extansively, for example, speech rate, and highlighting text under the mouse pointer.
- For Braille display the individual purchasing of a Braille display is required. The program has enhanced opportunities to maximize accessibility, ysing braille.
- The braille display supports six point braille and eight point computer braille.
- Window-Eyes now works too, in a limited way, if the video driver software on the computer is missing.
- The most common application software such as MS Office, Adobe Reader, Internet Explorer and Firefox works well with Window-Eyes. A current list is available on the website.
- A version of the program is available on a USB stick. Such version is mobile and therefore usable on different computers.
- A wide range of braille displays is supported via a serial, USB or Bluetooth connection.
-Available versions: Standard Edition for home use and Professional version in work situations where server versions of Windows are used and network support is required.

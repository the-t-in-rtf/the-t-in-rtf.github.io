Name Audio Notetaker
Description
  • Audio Notetaker is software to create which also notes, images and color codes can be added sound recordings (classes, meetings, interviews).
    - Students, teachers and professionals can manage their recordings and processing. It offers an alternative to written notes.
    - Sound recordings are displayed as blocks, for a good overview and easy navigation.
    - Use the arrow keys you can jump from one to the other fragment.
    - The software automatically splits the sound file into pieces and bring it up on the screen bars. With these bars you can go to work: you can emphasize the structure by providing, to highlight important sections and removing the less important pieces from them by colors.
    - Unwanted sound clips may be removed.
    - When recording a Skype call, you can then see who said what.
    - Data is recorded using the (built-in) microphone or you can record the system sound on your PC.
    - Presentations is possible and they can be subsequently exported to iTunes.
    - In addition to a sound the user slides (ppt), PDF files or images.
    - References to other documents or information sources via 'additional info' added.
    - Text of a presentation can be copied to the 'extra-Extended Info box.
    - Sorting and search functions are available.
    - Recordings of a voice recorder can be imported. Listen before importing is possible.
    - There is a quick start guide and a manual available.
    - Audio Notetaker is available in a Windows and Mac version. The Windows version requires at least Windows XP. The Mac version requires an Intel processor and Mac OS 10.5.8 Leopard or later.
    - A Dutch version is available. A French and German versions will follow later (end of 2014).

Name Mobile Speak for Symbian Phones 248455
Description

Mobile Speak is a screen reader application that you can install on certain mobile phones. It allows you to use the device, even if you can not read the visual screen. Information displayed on the screen is handled by text-to-speech software to synthetic speech output so that you may listen through the speakers of your phone or via a headset. The screen content can be presented in Braille if the mobile phone connections with a braille reading line. Speech and Braille output can simultaneously or independently of each other, are used for the following functions to perform:
- Receive calls and perform, and be informed of missed calls
- Read the identification of a caller when an incoming call, and check the number you have entered before you call
- Manage your contacts and call logs'
- Check the battery status, signal strength, bluetooth settings, and other details of the status
- Create and read text messages (SMS), multimedia messaging (MMS) and e-mail
- Surf the Internet and fill in online forms
- Read different types of files including Microsoft Word and simple text
- Read text by character, word or line
- Plan events and tasks
- Check the date and time
- Set the clock and alarm
- Create text and voice notes
- Perform arithmetic operations
- Convert from one currency or unit of measurement to another
- Listen to stored media files or internet audio streams
- Browse the system files and folders on the phone and memory card
- Configure phone settings, themes, profiles and ringtones
- Assign speed dial keys and voice tags for
- Access to other integrated 3rd-party applications such as Fringe or IM Plus instant messenger, Wallet, and Wayfinder Access for GPS navigation, to name but a few
- Connect to other mobile phones but also laptops and computers to exchange information
Mobile Speak, the software of your phone on, that way you still have access to the built-in functions and applications. In fact, so can compatible 3rd-party programs to install more to do with your mobile phone. External hardware such as Bluetooth keyboards and headsets are also supported similar, so you wireless control over the phone.
More information can be found on the website www.mobilespeak.com

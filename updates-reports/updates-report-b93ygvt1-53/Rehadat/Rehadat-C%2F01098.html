Product Type:
Input modification for mouse control with single-touch buttons

Areas of Application:
- enables the control of a standard computer mouse and important additional functions even with minimal motor skills
- with pronounced cerebral or central movement disorders, neurological disorders or neuromuscular disabilities, e.g. In ALS, spastic and flaccid paralysis
- allows mouse control via individual buttons / sensors

Description:
For the movement of the mouse pointer with a key, the desired target position is determined in two steps. The control of the mosaic keys is carried out via the enclosed keyboard emulation DIPAX ClickMaster.

How Mouse Motion Works:
When the ScanBox is running, a vertical bar moves permanently from left to right and back (scanning). If the bar is above the desired click position, a sensor must be pressed, whereupon the bar no longer moves. A second, horizontal bar is then displayed, which moves permanently from top to bottom and back again. If this bar is also above the desired click position, a sensor must be pressed again. A previously preselected mosaic function is applied immediately to the intersection thus determined.

How it works:
In a menu on the screen, buttons are displayed for the mouse button functions: Click Left mouse button, Right mouse button, Click Middle mouse button, Click and drag left, Double click left. After pre-selection of one of these mosaic function buttons, the corresponding mosaic function is triggered automatically.
trigger:
- USB-to-PC connection button included in delivery
- optional connection of an external sensor via a mini-jack socket possible

Features / Components:
- A standard mouse can be used in parallel.
- Switch between any open programs (windows) possible.
- Scan area automatically adapts to the size of the active window.
- Customer-specific extensions / adaptations possible.
- Contains a menu (click menu) for selecting a Mapping action.
Click menu functions:
- Mapping button functions: Click left, right, center; Click and drag; Double-click.
- Menu can be displayed horizontally, vertically, or double row
- different display sizes (zoom) adjustable.

System requirements:
- IBM compatible PC
- USB interface
- Windows Vista / 7/8/10

Scope of Delivery:

- CD

Product Type:
Screen Reader

Description:
The software has a screen magnifier, a screenreader as well as a support for Braille lines. SuperNova provides people with blindness or visual impairment with independent access to their computers for work and education.
The various functions are also available individually: enlargement of the screen, magnification with pre-reading function for texts, screenreader
In addition, there are all variants on mobile USB sticks.

Features / Components:
Enlargement:
- Magnification from 1.2x to 64x
- gradual enlargement
- excellent fonts smoothing (True Fonts)
- Multiple monitor support
- 8 magnification views
- Line mode
- Doc Reader
- Fixed areas

With SuperNova Connect & View, you can connect any High Definition (HD) camera to the computer to display magazines, newspapers, invoices, and other items on the screen

Speech:
- High-quality, human sounding speech synthesis
- Selection between character echo, word echo, etc.
- Adjustable voice speed and volume
- Text Style Announcement
- Announcement from the text under the mouse
- Read from here - tool
- Click and read from here
- Language when completing forms on web pages
- Language during the login process
- OCR function to scan and read mail, manuals, and more

Screen Reader:
- Dolphin Cursor
- Quick navigation buttons
- Object finder
- predefined verbosity settings
- Generate verbosity settings

Braille:
- Supports a large number of serial, USB & Bluetooth Braille lines
- shorthand and inscription, computer & literature Braille
- Supports a physical & logical Braille mode
- Braille input
- various Braille settings
- Display of Braille on the monitor

Compatibility:
- SuperNova can be installed in Wide and Local Area Network (WAN, LAN)
- Windows 10
- Windows 8.1
- Windows 7 (32/64 Bit Edition)
